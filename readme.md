## デザイン
元々は、ホテルのサイトとして作られました。

基本的なアイデアは案内サイトに変わってきましたが、案内サイトがホテルサイトのように見えるというアイデアが好きだったので、ホテルのコンセプトで開発を続けました。その点で、よりクリエイティブだと思います。

デザインはアマンホテルを参考にしています。
